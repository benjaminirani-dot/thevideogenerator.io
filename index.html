Note: This prototype needs backend endpoints:
  • /api/generate-script → accepts JSON {topic, duration, style} and returns {script}
  • /api/generate-video → accepts JSON {script, music, duration, style} and returns a video file (binary) — or a URL to the video.
The frontend doesn't enforce limits — backend should implement quota/usage rules or billing if you want true unlimited control.
